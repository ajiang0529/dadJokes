Team Members: Andy Jiang (ajiang25) Sharon Wong (swong132)

Description: Simple web application for random generation, 
deletion, and saving of dad jokes. Dad jokes are saved to MongoDB 
where you can then retrieve them (table format) or delete them.
To run, do node jokesServer.js <PORTNUMBER>

API Links: https://www.dadjokes.io/documentation/getting-started
